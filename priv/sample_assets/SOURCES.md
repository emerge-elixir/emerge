Sample asset sources (retained for docs examples and tests):

- `static.jpg`: https://picsum.photos/id/1018/640/420
- `fallback.jpg`: https://picsum.photos/id/1043/640/420
- `tile_quad.svg`: generated in-repo as a 2x2 test pattern
- `template_cloud.svg`: adapted from https://github.com/tabler/tabler-icons/blob/master/icons/outline/cloud.svg

These files are committed so root docs examples and renderer tests can run offline.

Tabler icons are MIT licensed. The local SVG copies keep the same geometry with sample-specific stroke colors.
The MIT license text is vendored at the repo root in `licenses/Tabler-icons-MIT.txt`.

The JPEG placeholders are retained for offline docs/tests use and are not included in the Hex package.
